## ISS

**A cli which lets you know the current position of the International Space Station, as well as who is on board and other interesting random trivia **


USAGE:

| Options                   | Description                                  |
|--------------------------:|----------------------------------------------|
| --                        |                                              |
| --                        |                                              |
| --version                 | Show the version.                            |
| --help                    | Show help information.                       |
